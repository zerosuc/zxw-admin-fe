## 说明

#### npm i

#### npm run dev

### 打包

npm run build:prod

#### 技术栈

- 语言：Golang、Vue+Typescript；
- 后端：gin gorm 、Casbin；
- 前端：Vue3、Vite、Element-Plus、TypeScript、Pinia 等；

- 起来mysql和redis
- go run main.go
